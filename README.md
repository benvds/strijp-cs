# [Strijp-CS website](http://strijpcs.nl)

## Quick start

### Requirements

1. [Node.js](http://nodejs.org)
2. [HTML5Boilerplate node build script](https://github.com/h5bp/node-build-script)

### Developing locally

    git clone git://github.com/benvds/strijp-cs.git
    cd strijp-cs
    h5bp server

### Building

    h5bp build
